Privacy Policy

Effective Date: April 6, 2026

1. Data Collection

This application ("Sandbox") accesses Oura Ring data via the Oura API v2. Specifically, it requests access to Sleep, Activity, Readiness, and Resilience scores.

2. Data Usage

The collected data is used exclusively for personal visualization within a private Pigment workspace. Data is processed via a private script and is not shared with any third parties other than Pigment.

3. Data Storage

OAuth tokens (Access and Refresh tokens) are stored securely in a private environment (e.g., GitHub Secrets or AWS Vault). No health data is stored permanently outside of the destination Pigment account.

4. User Rights

As the sole user of this application, I reserve the right to revoke API access at any time via the Oura Cloud settings.
